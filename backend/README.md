## Backend Note Nest

The backend of the Note Nest Application

## 🛠️ Setup

1. Clone the repo (if not done yet):

```
git clone https://github.com/Yusufdot101/Note-Nest.git
cd Note-Nest/backend
```

2. Run docker-compose:

```
docker-compose up
```
